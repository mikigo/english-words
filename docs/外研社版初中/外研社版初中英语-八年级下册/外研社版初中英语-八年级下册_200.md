## 200. prefer

**更喜爱；钟爱**

:::tip{title=🤩短语}

- prefer to do （宁可做某事；更喜欢做某事）

:::

:::

:::warning{title=🤔同义词}

- introduce, promote, present, upgrade, file （更喜欢；宁愿；提出；提升）

- affect, take a fancy to （喜欢；愿意）

:::
