- on the bit （(马)老老实实上着嚼子的，(马)被勒紧缰绳行走）

- drill bit （钻头，钎头；强钻头）

- diamond bit （金刚石钻头；钻石钻头）

- wait a bit （[口语]稍等一等，等一下）

- rock bit （牙轮钻头；岩石锧头）

- a bit off （[英口]有点失常；不太对劲）

- a bit on （[俚语]醉(一般与get连用)）

- stop bit （n. 停止位）

- roller bit （牙轮钻头；旋转钻头）

- bit stream （[计]位流，比特流）

:::

:::note{title=🎤例句}

:::
