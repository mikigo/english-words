**推动，增加；对…施加压力，逼迫；按；说服**

**推，决心；大规模攻势；矢志的追求**

:::tip{title=🤩短语}

- push for （奋力争取）

- push up （增加；提高；向上推）

- push on （推进；推动；努力向前）

- push through （完成；挤著穿过）

- push ahead （向前推进）

- push back （把…向后推；推回）

- push rod （制动缸推杆；推棒；顶杆）

- push out （拉出，排出；推出去）

- push in （推进，闯入）

- push button （按钮）
