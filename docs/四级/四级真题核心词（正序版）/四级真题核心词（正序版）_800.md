## 800. visible

`'vizəbl`  **看得见的**

:::tip{title=🤩短语}

- visible light （可见光）

- visible spectrum （可见光谱）

- visible image （可见象；可见像；视像）

- visible range （可见范围，可见光谱区）

- visible trade （有形贸易）

- visible laser （可见激光；可见激光器；可见镭射）

:::

:::note{title=🎤例句}

:::

:::warning{title=🤔同义词}

- obvious, distinct, seen, evident （明显的；看得见的；现有的；可得到的）

:::
